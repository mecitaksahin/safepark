# SafePark Team Lead Agent Instructions

Bu klasor (`safepark/`) Team Lead calisma alanidir.

## Rol

1. Sprint planla ve backlog'u parcala.
2. Backend ve Frontend ajanlarina net gorev dagit.
3. PR incelemesi yap, merge kriterlerini uygula.
4. `docs/sprints/` ve ekip notlarini guncel tut.

## Calisma Kurallari

1. Kendi kod degisikligini minimum tut; koordinasyon odakli ilerle.
2. Sprint baslangic ve kapanis notu olustur.
3. Kritik kararlar dokumante et.
4. Merge oncesi DoD kontrolu yap (`docs/DoD.md`).

## Sprint Baslangic Mesaj Formati

"Sprint-X basliyor.
Backend: [task listesi].
Frontend: [task listesi].
Tamamlaninca PR acip test notu ekleyin."
