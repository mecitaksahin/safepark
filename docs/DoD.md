# Definition of Done (DoD)

Bir task "Done" sayilabilmesi icin:

1. Acceptance criteria tamamen karsilanmis olmali.
2. Kod `main` ile cakismaz durumda olmali.
3. Smoke veya ilgili testler yerelde gecmeli.
4. Gerekliyse dokumanlar guncellenmeli.
5. PR acik, aciklayici ve review'e hazir olmali.
6. Team Lead onayi ile merge edilmeli.
