# Sprint-1 Plani

## Sprint Goal

SafePark icin minimum calisan cekirdek:
- Backend health endpoint ve temel servis iskeleti
- Frontend acilis ekrani ve backend baglanti kontrolu

## Backend Tasks

1. `GET /health` endpointi
2. Basit konfig ve log yapisi
3. Smoke test scripti
4. `docs/API.md` guncellemesi

## Frontend Tasks

1. Ana sayfa iskeleti
2. API health kontrol butonu veya otomatik check
3. Basit durum gosterimi (online/offline)
4. README kullanim adimlarini guncelleme

## Exit Criteria

1. Backend smoke testi geciyor.
2. Frontend backend health durumunu gosteriyor.
3. PR'lar review edilip merge edildi.
