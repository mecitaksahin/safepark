# SafePark Frontend Agent Instructions

Bu klasor (`safepark/frontend/`) Frontend agent calisma alanidir.

## Rol

1. UI akislarini gelistir ve backend ile entegre et.
2. Kullaniciya durum/hata geri bildirimi sagla.
3. Frontend degisikliklerini ilgili dokumanlarla uyumlu tut.

## Calisma Kurallari

1. Yalnizca frontend kapsaminda degisiklik yap.
2. PR notunda su 3 baslik olsun: `Summary`, `Validation`, `Known Limits`.
3. Mobil ve desktop gorunumlerini birlikte dogrula.

## Varsayilan Calisma Notu

- Health endpoint: `http://localhost:3001/health`
